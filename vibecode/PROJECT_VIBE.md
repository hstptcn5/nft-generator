# PROJECT VIBE & MISSION

## CORE PURPOSE
Build onchain applications (DApps and Mini Apps) on Base - Ethereum Layer 2 blockchain that enables fast, low-cost, builder-friendly development. Base brings the next billion users onchain with sub-second, sub-cent transactions and comprehensive developer tools.

## TARGET USERS
- Web3 developers building DApps on Base
- Developers creating Mini Apps for social distribution
- Teams integrating Base Account for authentication and payments
- Builders needing gasless transactions and smart wallet features

## KEY FEATURES
- OnchainKit integration for React components and utilities
- Base Account (Smart Wallet) for universal sign-on and USDC payments
- MiniKit SDK for building Mini Apps in Base App
- Paymaster integration for gasless transactions
- Smart contract deployment with Foundry
- Real-time wallet connections via Wagmi/Viem

## SUCCESS METRICS
- Successful deployment of smart contracts to Base Sepolia/Mainnet
- Working wallet connection and transaction flows
- Gasless transactions functioning via Paymaster
- Mini App discoverable and working in Base App
- Base Account authentication and payments operational

## TECHNICAL VIBE
- Prefer: Real integrations, no mocks, professional code style
- Avoid: Mocks, emojis in code, overly clever naming
- Style: Clean, readable, production-ready code
- Network: Base (Ethereum L2) - Mainnet and Sepolia testnet
- Framework: Next.js with App Router
- Web3: Wagmi + Viem for blockchain interactions
